# Uma análise demográfica da Bahia entre 2000 e 2024 📊
Este trabalho apresenta uma análise demográfica do estado da Bahia entre os anos de 2000 e 2024, investigando as principais transformações em sua dinâmica populacional, estrutura etária, fecundidade, mortalidade, urbanização e processos migratórios. Utilizando dados do Censo Demográfico do IBGE, DataSUS e SIDRA. Foi desenvolvido para a disciplina de Demografia da Universidade Federal de Goiás.

## 🛠️ Tecnologias e Bibliotecas
- R
- RStudio
- readxl
- dplyr
- ggplot2
- tidyverse
